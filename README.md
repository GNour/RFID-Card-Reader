# RFID-Card-Reader
RFID Card Reader built using Arduino uno, Nodemcu, RFID-Reader RC522, and LCD


## Connections

### Nodemcu with RFID RC522
|Nodemcu ESP8266|RFID RC522|
|------------|------------|
|D2|SDA/SS|
|D5|SCK|
|D7|MOSI|
|D6|MISO|
|GND|GND|
|D1|RST|
|3V/3V3|3.3V|

###  Arduino uno with LCD
|LCD|Arduino uno|
|------------|------------|
|D4|5|
|D5|4|
|D6|3|
|D7|2|
|E|11|
|RS|12|
|A|5V|
|K|GND|
|VDD|5V|
|VSS|GND|
|V0|6|
|RW|GND|
